Line-height Inheritance

When line-height is set by percentage, MacIE does not make the calculated value of the property inherit, rather it makes the percentage value itself inherit. This causes lines to change height when they should not.

1. Line-Height at 130%

The following paragraph contains two sizes of text. However, because the line-height is set by using percentages, the height of every line should be exactly the same, that is, 13px.

   .test       { font-size: 10px; line-height: 130%; }
   .bigwords {font-size: 300%; vertical-align: bottom; }
