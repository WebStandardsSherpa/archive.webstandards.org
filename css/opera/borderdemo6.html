Borders on Inline Elements Invade Line Spacing

This paragraph contains a number of inline EM elements. These elements should have a yellow background and a thin solid black border, but this background should not extend into the inter-line spacing. Opera extends each background half the leading (which is the line-height minus font-size) above and below the text. According to the spec, box properties and line-height on inline elements do not affect each other.
